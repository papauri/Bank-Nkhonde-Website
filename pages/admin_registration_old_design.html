Group Details

MWK
Initial contribution required when joining. Same amount for all members. Can be paid in full or within 2 months (configurable), but must be fully paid.
MWK
Regular monthly payment amount. Same for all members in the group. Due on the same day each month (based on cycle start date).
% %
Interest rate (%) charged on the full loan amount in the 1st month of repayment. Example: 15% on MWK 400,000 = MWK 60,000 interest.
% %
Interest rate (%) for 2nd month, calculated on remaining loan balance. If different from Month 1. Leave empty to use Month 1 rate.
% %
Interest rate (%) for 3rd month and beyond, calculated on remaining balance. If different from previous months. Leave empty to use previous rate.
%
Percentage penalty added on top of missed loan payment amount if payment deadline is missed. Same rate for all members. Example: 10% penalty on MWK 200,000 = MWK 20,000 extra.
%
Percentage penalty added to missed monthly contribution amount if payment is late. Same rate for all members. Example: 5% penalty on MWK 500,000 = MWK 25,000 extra.
